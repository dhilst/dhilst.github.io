title: How to install vim 8 Centos 7.
tags: [vim, centos]
---

There is a package in [COPR](https://copr.fedorainfracloud.org/) for this:

```
yum install yum-plugin-copr
yum copr enable hnakamur/vim
yum instal vim
```
